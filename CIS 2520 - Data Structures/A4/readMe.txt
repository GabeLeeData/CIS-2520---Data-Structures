***Compiling***

To compile the program simply type "make" within the terminal of the appropriate directory where all the files are located.
After the program is compiled type in "./avltree" to run the program.   
A example of running the program is "./avltree" within the terminal after "make" is typed into the terminal. 



***Programs***

avltree. 

When launching the program, the user will be presented with 7 options. Each option can be selected from the menu by inputing the corresponding number such as 1,2, or 3. 
The program will launch the corresponding function with the number selected. The program can read in the text file, and implement a AVL Tree. The program can find keys,
insert keys, balance the tree after insertion, remove keys, finding the stats of the tree and finding keys that are above a certain frequencie. 


****Limitations***

Limitations for this program are that the total Count for selection 5 may not be implemented correctly.
